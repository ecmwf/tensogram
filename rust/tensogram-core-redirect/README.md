# tensogram-core

This crate has been renamed to [`tensogram`](https://crates.io/crates/tensogram).

It is kept on crates.io as a thin re-export crate so existing
dependents can upgrade at their own pace. New code should depend on
`tensogram` directly:

```bash
cargo add tensogram
```
